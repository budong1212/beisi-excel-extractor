# 贝思Excel数据提取工具

从 Excel 文件的指定列中，提取每个单元格值的第 N 位到第 M 位字符，并保存为原格式文件。

## ✨ 功能特性

- 📂 支持拖拽方式加载文件
- 📋 支持多文件同时批量处理
- 🔢 可指定目标列（如 A、B、AA 等）及字符起止位置
- 🏷️ 可设置跳过表头行、跳过表尾行
- 💾 输出保留原 Excel 格式（.xlsx/.xls/.xlsm）
- ⚡ 实时显示处理速度与进度条
- 🔒 输出目录与原文件同名时自动加序号，防止覆盖

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python main.py
```

## 📦 打包为可执行文件（Windows）

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "贝思excel数据提取" main.py
```

打包完成后，`dist/贝思excel数据提取.exe` 即为独立可执行文件，无需安装 Python。

## 🖥️ 界面说明

| 区域 | 说明 |
|------|------|
| 源文件 | 拖拽或点击按钮选择一个或多个 Excel 文件 |
| 目标列 | 填写列字母，如 `A`、`C`、`AB` |
| 起始位 / 结束位 | 提取第几位到第几位（从1开始计数） |
| 首行为表头 | 勾选后跳过第一行不处理 |
| 有表尾 | 勾选后设置末尾跳过的行数 |
| 输出位置 | 选择处理后文件的保存目录 |
| 处理进度 | 显示进度条、当前文件名及处理速度 |

## 📝 示例

数据列值为 `123456789455`，设置提取第 **3** 位到第 **8** 位，结果为 `345678`。

## 🛠️ 环境要求

- Python 3.8+
- Windows / macOS / Linux
